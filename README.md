## Можливості

- Паспортні криві потужності всіх чотирьох моделей Dragon (19 точок RPM→W)

- Вкладка «Модель» з анімованими SVG-схемами для пояснення фізики клієнту

## Фізичні моделі

- **Jensen (1983)** — wake-слід з лінійним розширенням конуса
- **Frandsen (2006)** — вплив турбулентності на розсіювання wake: `kw_eff = kw + 0.5 × TI`
- **Von Mises** — статистична модель рози вітрів з параметром концентрації κ
- **Weibull** — розподіл швидкостей вітру для розрахунку середньорічної генерації
- **Dabiri (Caltech, 2011)** — blockage-ефект для VAWT ферм

Точність моделі: ±20–30% (pre-feasibility). Для проєктних рішень рекомендується анемометрування майданчика.
